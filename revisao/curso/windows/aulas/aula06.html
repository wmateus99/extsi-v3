Exercício: Usando o Microsoft Edge para Navegar e Pesquisar na Web

Nesta aula, você aprenderá a usar o navegador Microsoft Edge para navegar na web, pesquisar por conteúdo, organizar seus favoritos e personalizar sua experiência de navegação.

O objetivo deste exercício é explorar as funcionalidades do Microsoft Edge, incluindo a configuração de página inicial, o uso de buscadores e a organização de favoritos.

Passo 1: Abrindo o Microsoft Edge

  1. Abra o Menu Iniciar e pesquise por Microsoft Edge.
  2. Clique no ícone do Edge para abrir o navegador.
  3. Observe a interface do navegador. Você verá a barra de endereços, onde você pode digitar sites, e uma área para visualizar suas abas abertas.

Passo 2: Configurando a Página Inicial do Microsoft Edge

  1. No Microsoft Edge, clique no ícone de três pontos no canto superior direito e selecione Configurações.
  2. Na seção Ao iniciar, selecione Mostrar página de Nova Guia
  3. Agora, personalize sua página de Nova Guia escolhendo entre as opções Em Foco, Inspirador, Informativo ou Personalizado.
  4. Experimente escolher a opção Inspirador e observe como o fundo da página inicial muda para incluir uma imagem e notícias.

Passo 3: Pesquisando na Web

  1. Na barra de pesquisa do Microsoft Edge, digite "Tecnologia" e pressione Enter.
  2. Observe os resultados de pesquisa fornecidos pelo Bing. Clique em um dos links para ler mais sobre o assunto.
  3. Experimente também a pesquisa por voz. Clique no ícone de microfone ao lado da barra de pesquisa e diga algo como "Notícias sobre inteligência artificial".

Passo 4: Organizando Favoritos no Microsoft Edge

  1. Após abrir uma página de interesse, clique no ícone de estrela ao lado da barra de endereços para adicionar a página aos seus Favoritos.
  2. Escolha a pasta onde deseja salvar o favorito ou crie uma nova pasta para organizar melhor seus sites preferidos.
  3. Abra o Menu de Favoritos clicando novamente no ícone de estrela e veja os sites salvos. Explore as opções para gerenciar seus favoritos, como renomear ou excluir.

Passo 5: Usando o Modo Leitor no Microsoft Edge

  1. Abra um artigo de sua escolha no Microsoft Edge.
  2. Clique no ícone de livro na barra de endereços para entrar no Modo Leitor.
  3. Observe como a página é reorganizada, destacando apenas o texto e removendo elementos desnecessários como anúncios e barras laterais.
  4. Experimente alterar o modo de visualização, ajustando o tamanho do texto e o fundo da página para torná-lo mais confortável para leitura.

Passo 6: Adicionando Extensões ao Microsoft Edge

  1. No Microsoft Edge, clique novamente no ícone de três pontos no canto superior direito e selecione Extensões.
  2. Na página de extensões, clique em Obter extensões da Microsoft Store.
  3. Pesquise por uma extensão, como "Grammarly" para correção gramatical, e clique em Adicionar ao Edge.
  4. Após a instalação, explore como a extensão funciona e personalize suas configurações, se necessário.

Passo 7: Personalizando o Navegador Microsoft Edge

  1. Volte para as Configurações do Edge e explore outras opções de personalização.
  2. Experimente alterar o tema do navegador, configurando o Edge para o modo Claro ou Escuro, conforme sua preferência.
  3. Explore as configurações de privacidade e segurança, ajustando a proteção contra rastreadores e o bloqueio de cookies.

Este exercício vai ajudá-lo a usar o Microsoft Edge de forma eficiente para navegar na web, organizar seus favoritos, melhorar sua experiência de leitura e personalizar o navegador conforme suas preferências.
