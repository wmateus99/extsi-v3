Revisão - Avançado

ATENÇÃO: Siga as regras

1ª Regra - Todas as tabelas, devem ter no mínimo 8 linhas.

2ª Regra - Não faça a tabela de qualquer jeito, faça formatações e ajuste para um melhor visual, pois o professor irá avaliar.

3ª Regra - Caso a primeria e segunda regra não sejam atendidas, o professor pode definir a atividade como "não concluída."

Aula 1: Revisão de Fórmulas e Funções

1. Abra o Excel e crie uma planilha com as colunas: Produto, Quantidade Vendida, Preço Unitário e Receita Total.

2. Insira os dados fictícios de 5 produtos: dê nomes aos produtos, como "Caneta", "Lápis", etc.; preencha as quantidades vendidas e os preços unitários.

3. Na coluna Receita Total, use a fórmula =Quantidade * Preço Unitário para calcular o total de vendas por produto.

4. Em uma célula separada, utilize a função SOMA para calcular o total geral das receitas.

5. Crie células adicionais para calcular:
• A receita média (MÉDIA),
• O menor valor (MÍNIMO),
• O maior valor (MÁXIMO).

6. Adicione outra coluna chamada "Desconto". Utilize a função SE para aplicar um desconto de 10% nas receitas maiores que R$ 300. Fórmula sugerida: =SE(Receita>Total, Receita*10%).

Aula 2: Funções de Texto

1. Monte uma planilha contendo os seguintes dados: Nome Completo, Cargo e Código do Funcionário (ex.: João da Silva, Gerente, 123456).

2. Usando a função ESQUERDA, extraia os primeiros nomes (ex.: João).

3. Utilize DIREITA para retornar os últimos 3 dígitos do código do funcionário.

4. Adicione uma coluna para concatenar o Nome Completo e o Cargo. Utilize as funções CONCAT ou UNIRTEXTO para criar uma frase como: "João da Silva é Gerente.".

Aula 3: Funções Lógicas

1. Crie uma tabela com os seguintes dados: Nome do Aluno, Frequência (%), Nota 1, Nota 2, Nota 3, Nota 4, Situação.

2. Calcule a média das notas usando a função MÉDIA.

3. Na coluna Situação, utilize a função SE combinada com E para determinar se o aluno foi aprovado. Condição: Média ≥ 7 e Frequência ≥ 75%.

Aula 4: Funções de Matemática e Estatística

1. Crie uma tabela com as colunas: Produto, Categoria, Preço e Quantidade Vendida. Preencha com ao menos 8 linhas de dados.

2. Adicione uma coluna para calcular a Receita de cada produto: =Preço * Quantidade.

3. Use a função SOMASE para somar as Receitas separadamente por Categoria.

4. Utilize CONT.SE para contar quantos produtos pertencem a uma categoria específica.

Aula 5: Funções Estatísticas

1. Monte uma tabela com dados de uma pesquisa: Nome, Gênero, Idade e se comprou (Sim ou Não).

2. Use a função CONT.VALORES para contar quantos participaram da pesquisa.

3. Utilize CONT.NÚM para contar quantas idades foram registradas e MÉDIA para calcular a idade média.

4. Descubra a maior idade entre os que compraram algo utilizando MÁXIMOSES.

Aula 6: Funções de Data e Hora

1. Crie uma planilha com as colunas: Cliente, Data do Pedido, Prazo de Entrega (em dias) e Data de Entrega.

2. Use a função HOJE para obter a data atual e DIATRABALHO para calcular a Data de Entrega.

3. Adicione uma coluna para verificar se o pedido está atrasado: SE(Data de Entrega > HOJE(); "Atrasado"; "No Prazo").

Aula 7: Auditoria de Fórmulas

1. Crie uma tabela com Receita e Despesas e calcule o Lucro (Receita - Despesas).

2. Use as ferramentas de rastreamento de precedentes e dependentes para identificar possíveis erros.

3. Aplique a função SEERRO para substituir erros nas fórmulas por "Valor Inválido".

Aula 8: Funções de Pesquisa e Referência

1. Monte uma tabela com os dados: Código, Nome do Funcionário, Departamento e Salário.

2. Use PROCV para buscar o Departamento e o Salário com base no Código do Funcionário.

3. Crie uma tabela auxiliar com aumentos salariais e utilize ÍNDICE e CORRESP para buscar os valores de aumento.

Aula 9: Tabela Dinâmica e Formatação Condicional

1. Crie uma tabela com os seguintes dados: Produto, Região, Mês, Quantidade Vendida e Receita.

2. Preencha a tabela com pelo menos 20 registros fictícios, variando os produtos, regiões e meses.

3. Insira uma Tabela Dinâmica que mostre a soma das Receitas por Região e Produto.

4. Adicione uma Formatação Condicional para destacar produtos cuja Receita seja maior que R$ 10.000.

Aula 10: Gráfico Dinâmico

1. A partir da tabela dinâmica criada na Aula 9, crie um Gráfico Dinâmico que exiba a Receita por Produto e Região.

2. Personalize o gráfico, alterando o título e adicionando rótulos de dados.

3. Adicione Segmentações para permitir filtros interativos, como por Região e Mês.

Aula 11: Utilizando Formulários

1. Crie uma planilha para registrar solicitações de suporte técnico com as colunas: Nome, Departamento, Data da Solicitação, Prioridade e Status.

2. Habilite a guia Desenvolvedor no Excel e crie um Formulário que permita inserir dados diretamente na planilha.

3. Utilize validação de dados para as colunas "Prioridade" (opções: Alta, Média, Baixa) e "Status" (opções: Aberto, Fechado).

Aula 12: Macros

1. Grave uma Macro que formate automaticamente uma tabela de vendas, aplicando bordas, preenchimento de cabeçalhos com cor e alinhamento centralizado.

2. Insira um botão na planilha para executar a Macro criada.

Aula 13: Solver e Funções Financeiras

1. Crie uma tabela para simular um financiamento com as colunas: Valor do Empréstimo, Taxa de Juros Mensal (%), Número de Parcelas e Valor da Parcela.

2. Use a função PGTO para calcular o valor da parcela com base nos dados fornecidos.

3. Utilize o Solver para ajustar o número de parcelas de forma que o valor total pago não ultrapasse R$ 50.000.
