Revisão - Essencial

Aula 1: Introdução ao Editor de Planilhas

Questão: Crie uma planilha que organize as despesas mensais de uma residência. Insira colunas para "Tipo de Despesa" (ex.: Aluguel, Luz), "Valor" e "Vencimento". Formate as células com:

  • Cor de preenchimento para os cabeçalhos.
  • Formato de moeda para os valores.
  • Data abreviada para as colunas de vencimento.

Aula 2: Operadores Matemáticos

Questão: Elabore uma tabela para calcular o preço final de produtos após a aplicação de impostos e descontos. Inclua colunas para "Preço Bruto", "% Imposto", "% Desconto" e "Preço Final". Utilize fórmulas para calcular o preço final com a seguinte lógica:

  • Preço Final = Preço Bruto + (Preço Bruto * % Imposto) - (Preço Bruto * % Desconto).

Aula 3: Subtração, Multiplicação e Divisão

Questão: Desenvolva uma planilha que simule a divisão de despesas entre amigos após um jantar. Adicione colunas para "Nome", "Valor Gasto" e "Divisão". A coluna "Divisão" deve calcular quanto cada amigo deve pagar proporcionalmente ao valor gasto.

Aula 4: Funções Básicas (Média, Mínimo, Máximo)

Questão: Monte uma planilha que registre as notas de 5 alunos em 3 provas. Inclua colunas para "Nome do Aluno", "Nota 1", "Nota 2", "Nota 3" e "Média Final". Utilize a função MÉDIA para calcular a média final e destaque o aluno com a maior nota usando a função MÁXIMO.

Aula 5: Formatação de Planilhas

Questão: Crie uma planilha com as vendas diárias de um vendedor durante uma semana. Insira colunas para "Dia da Semana", "Produto", "Quantidade Vendida" e "Valor Total". Formate a planilha com bordas, cores para os cabeçalhos e aplique o estilo moeda na coluna "Valor Total".

Aula 6: Gráficos no Excel

Questão: Registre os dados de vendas de 4 produtos ao longo de 6 meses em uma planilha. Crie um gráfico de colunas que mostre o total de vendas por produto.

Aula 7: Funções de Pesquisa

Questão: Monte uma tabela com os dados de funcionários (ID, Nome, Departamento e Salário). Utilize a função PROCV para buscar informações de um funcionário com base no ID informado.

Aula 8: Classificação e Filtros

Questão: Crie uma planilha com uma lista de 10 produtos, incluindo informações como "Nome", "Categoria", "Preço" e "Quantidade em Estoque". Classifique os produtos por preço em ordem decrescente e aplique um filtro para exibir apenas os produtos da categoria "Eletrônicos".

Aula 9: Validação de Dados

Questão: Construa uma planilha de registro de pedidos com colunas para "Número do Pedido", "Cliente", "Data" e "Status". Adicione validação de dados na coluna "Status" para que seja possível selecionar apenas as opções "Pendente", "Em Andamento" ou "Concluído".

Aula 10: Funções Lógicas

Questão: Crie uma tabela para verificar a aprovação de alunos em um curso. Insira colunas para "Nome", "Nota Final" e "Situação". Utilize a função SE para preencher automaticamente a situação com "Aprovado" se a nota for maior ou igual a 7, e "Reprovado" caso contrário.

Aula 11: Proteção de Planilhas

Questão: Monte uma planilha para controle financeiro com colunas para "Categoria", "Descrição", "Valor" e "Data". Proteja a planilha de modo que apenas as células de "Valor" possam ser editadas.

Aula 12: Importação de Dados

Questão: Importe um arquivo CSV contendo informações de vendas e organize os dados em uma nova planilha. Aplique formatação condicional para destacar vendas acima de R$ 1.000.

Aula 13: Introdução ao VBA

Questão: Grave uma macro que insira automaticamente a data e a hora atual em uma célula selecionada ao pressionar um botão.

Aula 14: Automatizando Relatórios

Questão: Crie um relatório automatizado que consolide os dados de vendas de várias planilhas em uma única tabela dinâmica. Insira um gráfico baseado na tabela dinâmica gerada.
